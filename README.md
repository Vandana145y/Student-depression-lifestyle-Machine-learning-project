# Student-depression-lifestyle-Machine-learning-project
## Project Name

## Student Depression Prediction System Using Machine Learning

# Project Description

The Student Depression Prediction System is an end-to-end Machine Learning and MLOps project developed to predict depression risk among students based on academic, lifestyle, and personal factors. The project includes data preprocessing, exploratory data analysis, feature engineering, model development, hyperparameter optimization, experiment tracking, model evaluation, and deployment. Multiple machine learning algorithms were trained and compared to identify the most suitable model for depression prediction. The final model was deployed using Streamlit, enabling users to receive real-time predictions through an interactive web interface.

## Objectives

Analyze factors influencing student depression.

Build predictive machine learning models for depression detection.

Compare multiple classification algorithms.

Optimize model performance using Optuna.

Track experiments and model versions using MLflow.

Evaluate models using standard classification metrics.

Deploy the best-performing model using Streamlit.

Provide an easy-to-use interface for real-time predictions.

Demonstrate an end-to-end Machine Learning and MLOps workflow.

## Dataset Information
